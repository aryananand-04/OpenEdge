# Screenshots Directory

Place your app screenshots here:

- `main_activity.png` - Main screen with camera feed
- `grayscale_mode.png` - Grayscale processing mode
- `blur_mode.png` - Blur processing mode
- `edge_mode.png` - Edge detection mode
- `webview_activity.png` - WebView screen

## How to Capture Screenshots

1. Run the app on a device/emulator
2. Navigate to each mode
3. Use Android Studio's screenshot tool: View → Tool Windows → Logcat → Camera icon
4. Or use ADB: `adb shell screencap -p /sdcard/screenshot.png`
